Conclusion

From this research I understand how Hash Functions can be used to provide a mean of file integrity and verification. I understand the different options that are available when deciding what type of Hash Function algorithm will be used to provide file integrity such as using a hash function that creates a large hash value such as SHA-1 which is slower than MD5 and significantly slower than Adler32. They provide a degree of difficulty for intruders to try change files without being noticed by the recipient.
